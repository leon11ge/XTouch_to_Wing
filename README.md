# XTouch_to_Wing
Simple Script for controlling Behringer Wing with a Behringer X-Touch Control over Network.

#Disclaimer

The Skript is funktionional. Yet missing is the description for setting up the RTP Midi Connection (on your Raspi for example)

#How-to-use:
- Connect X-Touch-Extender & Wing over Ethernet (using a switch)
- Set static Ip in Wing Remote Settings
- Set X-Touch-Extender settings (MCU, no DHCP)
- Edit Ip-Adress in the ruedigec_v7 file
- Restart all devices
